# 🎨 移动端主题定制配置

## 📋 概述

主题定制配置基于我们的设计系统（色彩、字体、间距）对 uView UI 和 Iconify 进行深度定制，确保第三方组件库与我们的设计规范保持完全一致，打造统一的视觉体验。

> **重要提示**：本文档中的所有颜色配置都支持从 [20个配色方案](./color/配色方案索引.md) 中选择。您需要根据项目需求选择合适的配色方案，并替换相应的颜色值。

## 🌈 色彩主题配置

### 配色方案选择与配置

> **第一步：选择配色方案**
> 请从 [配色方案索引](./color/配色方案索引.md) 中选择适合您项目的配色方案，然后将对应的颜色值替换到以下配置中。

### 基础色彩变量定义

```scss
// uni.scss - 全局样式变量文件
// 基于选择的配色方案进行配置

// === 主色系（需要根据选择的配色方案替换）===
// 示例：清新自然色彩体系
$primary-color: #247BA0;      // 深海蓝 - 主色调
$secondary-color: #70C1B3;    // 薄荷绿 - 辅助色
$accent-color: #FF1654;       // 活力红 - 强调色
$neutral-light: #F3FFBD;      // 柠檬黄绿 - 中性浅色
$neutral-dark: #B2DBBF;       // 淡雅绿 - 中性深色

// === uView UI 主题变量覆盖 ===
$u-primary: $primary-color;
$u-success: $secondary-color;
$u-warning: #FFB800;          // 通用警告色
$u-error: $accent-color;
$u-info: $primary-color;

// === 文字颜色（来自字体系统）===
$u-main-color: #1A1A1A;      // 主要文字
$u-content-color: #666666;   // 次要文字
$u-tips-color: #999999;      // 辅助文字
$u-light-color: #CCCCCC;     // 禁用文字

// === 背景和边框 ===
$u-bg-color: #FFFFFF;        // 主背景
$u-bg-color-light: #F5F5F5;  // 浅色背景
$u-border-color: #E5E5E5;    // 边框颜色

// === 间距（来自间距系统）===
$u-padding: 32rpx;           // 标准内边距

$u-gap: 16rpx;               // 标准间隙

// === 圆角（来自间距系统）===

$u-radius-small: 8rpx;       // 小圆角
$u-radius-large: 24rpx;      // 大圆角

$u-font-size-lg: 32rpx;      // 大字号
$u-line-height: 1.5;         // 标准行高
```

```javascript
    warning: {
```

### 图标组件封装

```vue
<!-- components/theme-icon/theme-icon.vue -->
<template>
  <iconify-icon 
    :icon="icon"
    :style="iconStyle"
    @click="handleClick"
  ></iconify-icon>
</template>

<script>
import { iconifyConfig } from '@/config/iconify-config.js'

export default {
  name: 'ThemeIcon',
  props: {
    icon: {
      type: String,
      required: true
    },
    theme: {
      type: String,
      default: 'default',
      validator: value => ['default', 'primary', 'secondary', 'accent', 'success', 'warning', 'error'].includes(value)
    },
    size: {
      type: String,
      default: 'md',
      validator: value => ['xs', 'sm', 'md', 'lg', 'xl'].includes(value)
    },
    color: {
      type: String,
      default: ''
    }
  },
  computed: {
    iconStyle() {
      const themeStyle = this.theme === 'default' 
        ? iconifyConfig.defaultStyle 
        : iconifyConfig.themeStyles[this.theme]
      
      return {
        color: this.color || themeStyle.color,
        fontSize: iconifyConfig.sizePresets[this.size],
        display: 'inline-flex',
        alignItems: 'center',
        justifyContent: 'center'
      }
    }
  },
  methods: {
    handleClick() {
      this.$emit('click')
    }
  }
}
</script>
```

## 🔧 配置文件整合

### 主题配置管理

```javascript
// config/theme-config.js - 主题配置管理
export const themeConfig = {
  // 当前主题
  currentTheme: 'light', // light | dark | auto
  
  // 主题切换
  toggleTheme(theme) {
    this.currentTheme = theme
    this.applyTheme(theme)
  },
  
  // 应用主题
  applyTheme(theme) {
    const root = document.documentElement
    
    if (theme === 'dark') {
      root.classList.add('dark-theme')
    } else {
      root.classList.remove('dark-theme')
    }
    
    // 保存主题设置
    uni.setStorageSync('theme', theme)
  },
  
  // 初始化主题
  initTheme() {
    const savedTheme = uni.getStorageSync('theme') || 'auto'
    
    if (savedTheme === 'auto') {
      // 跟随系统主题
      const systemTheme = uni.getSystemInfoSync().theme || 'light'
      this.applyTheme(systemTheme)
    } else {
      this.applyTheme(savedTheme)
    }
  }
}
```

## ✅ 使用指南

### 1. 快速开始
```javascript
// main.js - 应用入口
import { themeConfig } from '@/config/theme-config.js'

// 初始化主题
themeConfig.initTheme()
```

### 2. 组件中使用主题
```vue
<template>
  <view class="page">
    <!-- 使用主题化按钮 -->
    <u-button type="primary" size="large">
      主要操作
    </u-button>
    
    <!-- 使用主题化图标 -->
    <theme-icon 
      icon="mdi:home" 
      theme="primary" 
      size="lg"
    ></theme-icon>
  </view>
</template>
```

### 3. 主题切换
```javascript
// 切换到深色模式
themeConfig.toggleTheme('dark')

// 切换到浅色模式
themeConfig.toggleTheme('light')

// 跟随系统主题
themeConfig.toggleTheme('auto')
```
