# 🎨 Design System

## 📋 概述

欢迎来到 NiceEasyAI 设计系统！这是一个完整的设计系统库，旨在为产品设计和开发提供统一、专业、易用的设计规范和资源。

## 🌈 设计系统组件

### 🎨 [Color - 色彩系统](./color/)

完整的色彩设计系统，包含 20 个精心设计的配色方案，涵盖各种应用场景和设计风格。

#### 📊 色彩系统特性
- ✅ **20个标准化配色方案** - 涵盖暖色、冷色、中性色等各种色系
- ✅ **完整的5色体系** - 主色、辅助色、强调色、中性浅色、中性深色
- ✅ **详细的设计指导** - 风格描述、色彩特性、情感表达
- ✅ **实用的应用示例** - 按钮设计、文字层级、状态指示
- ✅ **专业的使用建议** - 搭配原则、最佳实践、避免事项
- ✅ **开发者友好** - CSS变量定义、快速参考代码
- ✅ **无障碍考虑** - 对比度检查、文字搭配建议

### 📝 [Typography - 字体与排版](./字体与排版.md)

完整的字体与排版系统，定义了应用中所有文本元素的视觉规范，确保信息层次清晰、阅读体验优良。

#### 📊 字体排版特性
- ✅ **多平台字体栈** - 移动端、桌面端、等宽字体的完整支持
- ✅ **标准化字号体系** - 基于 16px 基准的 9 级字号规范
- ✅ **字重规范** - 从 Light 到 Bold 的 5 级字重体系
- ✅ **行高与间距** - 针对不同文本类型的行高和间距规范
- ✅ **响应式排版** - 适配不同设备的字号调整策略
- ✅ **文本颜色层级** - 主要、次要、辅助等 5 级文本颜色
- ✅ **无障碍考虑** - 对比度要求和可读性优化

### 📐 [Spacing - 基础数值与间距](./基础数值与间距.md)

基础数值与间距系统，定义了应用中所有空间、尺寸和视觉效果的标准规范，确保界面元素布局协调。

#### 📊 间距系统特性
- ✅ **16rpx 网格系统** - 基于 16 的倍数的移动端间距体系
- ✅ **8级间距规范** - 从 8rpx 到 128rpx 的完整间距等级
- ✅ **圆角系统** - 8 级圆角规范，适用于不同组件场景
- ✅ **4层阴影体系** - 针对移动端优化的阴影层级
- ✅ **rpx 响应式** - 自动适配不同移动设备屏幕
- ✅ **组件应用示例** - 卡片、按钮、表单等组件的具体应用
- ✅ **CSS 变量支持** - 完整的 CSS 自定义属性定义

### 🎯 [Icons - 图标系统引用规范](./图标系统引用规范.md)

## 🚀 快速开始

### 1. 选择配色方案（重要）
**首先**，请访问 [配色方案选择指南](./配色方案选择指南.md) 从 20 个配色方案中选择适合您项目的方案。

### 2. 选择设计组件
根据您的项目需求，选择相应的设计系统组件：
- **色彩设计** → 访问 [Color 色彩系统](./color/)
- **文字排版** → 访问 [Typography 字体与排版](./字体与排版.md)
- **布局间距** → 访问 [Spacing 基础数值与间距](./基础数值与间距.md)
- **图标系统** → 访问 [Icons 图标系统引用规范](./图标系统引用规范.md)
- **组件库** → 访问 [Components 组件库引用规范](./组件库引用规范.md)
- **主题定制** → 访问 [Theme 主题定制配置](./主题定制配置.md)

### 2. 查看设计规范
每个组件都包含详细的设计规范和使用指导：
- 设计原则和理念
- 具体的应用示例
- 最佳实践建议
- 常见错误避免

### 3. 获取设计资源
- **设计师**：查看设计规范和视觉示例
- **开发者**：复制 CSS 代码和变量定义
- **产品经理**：了解设计理念和应用场景

### 4. 应用到项目
按照规范将设计系统应用到您的项目中，确保设计的一致性和专业性。

## 🎯 设计原则

### 1. 一致性 (Consistency)
- 统一的视觉语言和交互模式
- 标准化的组件和规范
- 可预测的用户体验

### 2. 可用性 (Usability)
- 直观易懂的设计
- 无障碍设计考虑
- 多平台适配

### 3. 可扩展性 (Scalability)
- 模块化的设计组件
- 灵活的配置选项
- 易于维护和更新

### 4. 美观性 (Aesthetics)
- 现代简约的视觉风格
- 和谐的色彩搭配
- 优雅的视觉层次
