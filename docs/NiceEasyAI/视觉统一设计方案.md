# 🎨 视觉统一设计方案

## 基于温暖粉橘色彩体系的移动端视觉设计规范

---

## 📋 方案概述

本视觉统一设计方案基于**温暖粉橘色彩体系**，专为移动端应用设计，涵盖美妆时尚、生活方式、女性向应用等场景。通过精心设计的配色系统、图标规范、组件库、数值间距和字体排版，确保整个应用具有统一、优雅、温馨的视觉体验。

### 设计理念
- **温馨优雅**：营造温暖舒适的视觉氛围
- **时尚精致**：体现现代女性的审美品味
- **亲和友好**：建立用户情感连接
- **专业可信**：保持品牌的专业形象

## 🌈 配色系统规范

### 核心色彩体系

#### 主色调（Primary Color）- 珊瑚粉
- **HEX**：`#FFCDB2`
- **RGB**：`rgb(255, 205, 178)`
- **色彩特性**：温柔、浪漫、亲和
- **情感表达**：温馨、甜美、舒适
- **应用场景**：主导航栏、重要按钮、品牌标识、主标题

#### 辅助色（Secondary Color）- 蜜桃橘
- **HEX**：`#FFB4A2`
- **RGB**：`rgb(255, 180, 162)`
- **色彩特性**：活力、温暖、友好
- **情感表达**：热情、积极、愉悦
- **应用场景**：次要按钮、标签页、进度条、图标

#### 强调色（Accent Color）- 玫瑰粉
- **HEX**：`#E5989B`
- **RGB**：`rgb(229, 152, 155)`
- **色彩特性**：温柔、浪漫、亲和
- **情感表达**：温馨、甜美、舒适
- **应用场景**：重要通知、状态提示、特殊按钮、强调元素

#### 中性色1（Neutral - Light）- 紫灰色
- **HEX**：`#B5838D`
- **RGB**：`rgb(181, 131, 141)`
- **色彩特性**：神秘、优雅、创意
- **情感表达**：高贵、想象、神秘
- **应用场景**：页面背景、卡片背景、输入框背景、浅色分割线

#### 中性色2（Neutral - Dark）- 深紫灰
- **HEX**：`#6D6875`
- **RGB**：`rgb(109, 104, 117)`
- **色彩特性**：神秘、优雅、创意
- **情感表达**：高贵、想象、神秘
- **应用场景**：主要文字、图标、边框、次要信息文本

### 色彩搭配规则

#### 主要配色组合
1. **主导配色**：主色 + 辅助色 `#FFCDB2` + `#FFB4A2`
2. **对比配色**：主色 + 强调色 `#FFCDB2` + `#E5989B`
3. **和谐配色**：辅助色 + 中性色 `#FFB4A2` + `#B5838D`
4. **平衡配色**：强调色 + 中性色 `#E5989B` + `#6D6875`

#### 功能色彩映射
- **品牌色彩**：`#FFCDB2`（珊瑚粉）
- **成功状态**：`#FFB4A2`（蜜桃橘）
- **警告状态**：`#FFB800`（橙黄）
- **错误状态**：`#FF4757`（红色）
- **信息状态**：`#FFCDB2`（珊瑚粉）

#### CSS变量定义
```css
:root {
  /* 主色系 */
  --primary-color: #FFCDB2;
  --secondary-color: #FFB4A2;
  --accent-color: #E5989B;
  --neutral-light: #B5838D;
  --neutral-dark: #6D6875;

  /* 功能色 */
  --success-color: var(--secondary-color);
  --warning-color: #FFB800;
  --error-color: #FF4757;
  --info-color: var(--primary-color);

  /* 文字色 */
  --text-primary: #1A1A1A;
  --text-secondary: #666666;
  --text-tertiary: #999999;
  --text-disabled: #CCCCCC;

  /* 背景色 */
  --bg-primary: #FFFFFF;
  --bg-secondary: var(--neutral-light);
```

## 🎯 应用指南

### 使用优先级
1. **颜色优先**：所有组件首先遵循配色系统
2. **间距一致**：严格使用16rpx网格系统
3. **圆角规范**：根据组件类型选择合适圆角
4. **字体层级**：严格按照字号体系使用
5. **图标统一**：统一使用Iconify图标库

### 主题切换支持
```css
/* 支持深色模式 */
@media (prefers-color-scheme: dark) {
  :root {
    --bg-primary: #1A1A1A;
    --bg-secondary: #2A2A2A;
    --text-primary: #FFFFFF;
    --text-secondary: #CCCCCC;
  }
}
```

### 无障碍设计
- **对比度**：所有文本与背景对比度≥4.5:1
- **字号限制**：最小字号20rpx，推荐28rpx
- **触摸区域**：最小点击区域88rpx×88rpx
- **颜色替代**：重要信息不仅依赖颜色区分

### 性能优化建议
- **图标优化**：按需加载Iconify图标
- **样式复用**：使用CSS变量减少重复代码
- **组件缓存**：合理使用组件缓存机制
- **图片优化**：使用适当的图片尺寸和格式
